Privacy Policy

Last updated: May 2025

✓ No data collected

Teamflect CallCue does not collect, transmit, store remotely, or share any personal information or user data of any kind.

What the extension does

CallCue is a Chrome extension for sales teams using HubSpot and LemList. It reads contact fields visible on the currently open page — such as timezone, phone number, country, and city — and uses that information to calculate whether the contact is within their local working hours right now. The result is displayed as a floating panel and status orb directly in the browser tab.

Data processing

  • All processing happens locally in your browser, entirely in memory.
  • No contact data, page content, or computed results are ever sent to any server.
  • No analytics, telemetry, or crash reporting is collected.
  • The extension does not make any outbound network requests.

Local storage

CallCue uses Chrome's storage API solely to remember your personal UI preferences — such as panel theme (light or dark), your SDR working hours, and orb position on screen. This data never leaves your device and contains no contact or personal information.

Permissions

  • activeTab — to read the contact fields rendered on the active HubSpot or LemList tab.
  • storage — to persist your UI preferences locally.
  • Host access (app.hubspot.com, app.lemlist.com) — required to inject the extension's content scripts into these single-page applications so contact field data can be read from the live DOM.

Third parties

No user data is sold, transferred, or disclosed to any third party for any purpose.

Children's privacy

This extension is intended for professional use by sales teams and is not directed at children under 13.

Changes to this policy

Install via Developer Mode

While the extension awaits Chrome Web Store approval, you can install it directly from the source folder in under a minute.

Requirements

  • Google Chrome (or any Chromium browser — Edge, Brave, Arc)
  • The CallCue extension folder provided by your team

Steps

  1. Open Chrome and navigate to chrome://extensions in the address bar.
  2. In the top-right corner of that page, toggle Developer mode on. A new toolbar will appear.
  3. Click Load unpacked in the toolbar that just appeared.
  4. In the file picker, navigate to and select the CallMate folder (the one that contains the manifest.json file). Click Select Folder.
  5. Teamflect CallCue will now appear in your extensions list with an orange phone icon. It is active immediately — no restart needed.
  6. Open HubSpot or LemList and navigate to any contact record. The CallCue orb will appear automatically.

Keeping it updated

When your team sends a new version of the folder, replace the old folder contents with the new ones, then go back to chrome://extensions and click the ↺ refresh icon on the CallCue card. The updated version loads instantly.

Pinning the extension icon

Click the puzzle-piece icon ⊞ in the Chrome toolbar, find Teamflect CallCue, and click the pin icon beside it. This keeps the CallCue popup one click away in your toolbar.
