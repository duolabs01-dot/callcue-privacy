What the extension does

CallCue is a Chrome extension for sales teams using HubSpot and LemList. It reads contact fields visible on the currently open page — timezone, phone number, country, city — and uses that information locally to calculate whether the contact is within their working hours right now. The result is shown as a floating status orb and panel in your browser tab.

Data processing

  • All computation is performed locally in your browser, entirely in memory.
  • No contact data, page content, or computed results are sent to any server.
  • No analytics, telemetry, or crash reporting is collected.
  • The extension makes no outbound network requests.

Local storage

CallCue uses Chrome's storage API solely to save your UI preferences — panel theme, SDR working hours, and orb position. This data never leaves your device and contains no contact or personal information.

Permissions

  • activeTab — to read contact fields rendered in the active HubSpot or LemList tab.
  • storage — to persist your UI preferences locally on-device.
  • Host access (app.hubspot.com, app.lemlist.com) — required to inject content scripts into these single-page applications so contact field data can be read from the live DOM.

Third parties

No user data is sold, transferred, or disclosed to any third party for any purpose.

Children's privacy

This extension is intended for professional use by sales teams and is not directed at children under 13.

Install

Developer Mode Install

While the extension awaits Chrome Web Store approval, install it directly from the zip your team shared — takes under two minutes.

Requirements

  • Google Chrome (or any Chromium browser — Edge, Brave, Arc)
  • The CallCue zip file shared by your team lead

Steps

  1. Extract the zip. Right-click the zip file → Extract All. Save the extracted folder somewhere permanent — Desktop or Documents works. Don't move or delete it after installing.
  2. Open Chrome and go to chrome://extensions.
  3. In the top-right corner, enable Developer mode using the toggle. A new toolbar will appear at the top.
  4. Click Load unpacked in that toolbar.
  5. In the file picker, select the folder you extracted — it should contain a file called manifest.json. Click Select Folder.
  6. Teamflect CallCue will appear in your extensions list. Active immediately — no restart needed.
  7. Click the puzzle-piece icon ⊞ in your Chrome toolbar → find CallCue → click the pin icon to keep it visible.
  8. Open HubSpot or LemList and go to any contact record. The CallCue orb will appear automatically.

Updating to a new version

When a new zip is shared, extract it to the same folder location, overwriting the old files. Then go to chrome://extensions and click the ↺ refresh icon on the CallCue card. The new version is live instantly.

Orb not appearing?

  • Make sure you're on a contact record page, not the dashboard.
  • Click the CallCue icon in your toolbar and hit Refresh.
  • Still nothing? Go to chrome://extensions → click ↺ on the CallCue card → reload HubSpot.
